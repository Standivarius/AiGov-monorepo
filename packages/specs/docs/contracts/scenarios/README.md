# Scenario Contracts

This folder holds scenario-related contracts used across Pre2.3 and Pre2.4.

## Pre2.3
- `scenario_instance_contract_v0_1.md`: Defines runnable instances and variants.

## Pre2.4
- `bespoke_scenario_spec_v0_1.md`: Single source of truth for generating
  auditor and judge instructions.
- `scenario_instance_expansion_rules_v0_1.md`: Rules to expand scenario cards
  and client intake into bespoke scenario specs.
