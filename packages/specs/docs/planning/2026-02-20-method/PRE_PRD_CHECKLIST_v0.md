# Pre-PRD Checklist v0

Use this checklist before any PRD drafting.

## A) Story Map
- [ ] 6-stage backbone explicitly listed
- [ ] walking skeleton slice defined
- [ ] deferred stories listed by stage

## B) Example Maps
- [ ] one example map exists per walking-skeleton story
- [ ] rules are concrete and testable
- [ ] examples are specific and reproducible
- [ ] unresolved blocker questions are zero

## C) Scenario Pack
- [ ] scenario ID and scope are fixed
- [ ] expected artifacts by stage are listed
- [ ] deterministic comparison policy exists
- [ ] evidence expectations are listed
- [ ] holdout scenario policy is defined

## D) Governance
- [ ] no raw CoT persistence rule is explicit
- [ ] open contested decisions are ADR stubs
- [ ] method stage gate status is recorded

## E) Exit Rule
- [ ] all A-D checks pass
- [ ] Marius approval recorded
- [ ] only then start `PRD-P` and `PRD-M`
